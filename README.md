## Prerequisites

- Flutter 2.2.3

## Install

```sh
flutter pub get
```

## Usage (Dev)
I strongly recommend to use VS Code w/ Dart & Flutter extensions in order to properly debug, but you can use commands like:
```sh
flutter run
```
